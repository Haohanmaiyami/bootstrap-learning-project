# 🌐 Bootstrap Layout Practice

Simple training project with static HTML pages styled using [Bootstrap](https://getbootstrap.com/).  
This repo was created to practice basic layout, navigation and Bootstrap components.

> ⚠️ Note: this is a **frontend-only** project — no backend, database or API here.

---

## 📄 What’s inside

- `templates/index.html` – main page.
- `templates/catalog.html` – catalog page layout.
- `templates/category.html` – category page template.
- `templates/contacts.html` – simple contacts page.
- `main.py` – optional helper script (pages work as plain HTML without it).

All pages are built with:

- Bootstrap grid and basic components;
- responsive layout for desktop and mobile;
- simple navigation between pages.

---

## 🚀 How to run locally

You can either open the HTML files directly **or** run a tiny local server.

### Option 1 — open in browser

1. Clone the repository:
   ```bash
   git clone git@github.com:Haohanmaiyami/Bootstrap.git
   cd Bootstrap
   ```
2. Open `templates/index.html` in your browser (drag & drop or **File → Open**).

### Option 2 — run a simple HTTP server

From the project root:

```bash
cd templates
python3 -m http.server 8000
```

Then open in browser:

```text
http://127.0.0.1:8000/index.html
```

---

## 🛠 Tech stack

- HTML5  
- CSS3  
- Bootstrap 5 (via CDN)  
- (optional) Python 3 for local HTTP server

# 🌐 Практика вёрстки на Bootstrap

Небольшой учебный проект со статичными HTML-страницами, оформленными с помощью [Bootstrap](https://getbootstrap.com/).  
Репозиторий создан для тренировки базовой вёрстки, сетки и компонентов Bootstrap.

> ⚠️ Важно: это **только фронтенд** — без бэкенда, базы данных и API.

---

## 📄 Что внутри

- `templates/index.html` – главная страница.
- `templates/catalog.html` – страница каталога.
- `templates/category.html` – страница категории.
- `templates/contacts.html` – страница с контактами.
- `main.py` – вспомогательный скрипт (можно не использовать, страницы работают как обычный HTML).

На страницах используются:

- сетка Bootstrap;
- базовые компоненты;
- простая навигация между страницами;
- адаптивная вёрстка под десктоп и мобильные устройства.

---

## 🚀 Как запустить локально

Можно просто открыть файлы в браузере **или** поднять простой локальный сервер.

### Вариант 1 — открыть напрямую

1. Клонировать репозиторий:
   ```bash
   git clone git@github.com:Haohanmaiyami/Bootstrap.git
   cd Bootstrap
   ```
2. Открыть файл `templates/index.html` в браузере (через «Открыть файл» или drag & drop).

### Вариант 2 — запустить простой HTTP-сервер

Из корня проекта:

```bash
cd templates
python3 -m http.server 8000
```

Затем в браузере перейти по адресу:

```text
http://127.0.0.1:8000/index.html
```
